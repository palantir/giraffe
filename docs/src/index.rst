Installing
==========

Giraffe is published as a Maven-style artifact, but we recommend using
Gradle_ for builds:

.. code-block:: groovy

    repositories {
        mavenCentral()
    }

    dependencies {
        compile 'com.palantir.giraffe:giraffe-ssh:0.10.0'
        // or 'com.palantir.giraffe:giraffe-core:0.10.0' for local features only
    }

.. _Gradle: https://gradle.org/
